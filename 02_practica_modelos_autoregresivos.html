Modelos AR(1)

Un tipo de proceso estoc√°stico Markoviano es el modelo AR(1) o autorregresivo de orden 1.

\begin{cases} X(t_1) = 0 \ X(t_i)=X(t_{i-1})+(t_i), 0<||<1, i=2,3,‚Ä¶,T \end{cases}

Donde \(\epsilon(t_i)\sim \mathcal{N}(0,\sigma^2)\) es un proceso de ruido blanco Gaussiano

Una realizaci√≥n de este proceso estoc√°stico es una caminata aleatoria.

  • Codificar una funci√≥n proc_ar_1 que devuelva una realizaci√≥n de este proceso durante T pasos de tiempo. La funci√≥n debe recibir los par√°metros \(\alpha\) y \(\sigma\).

  • Generar una serie de tiempo para T=500 pasos de tiempo usando par√°metros \(\alpha=0.95\) y \(\sigma=1\). Eval√∫e combinaciones de estos par√°metros con \(\alpha=0.5\) y \(\sigma=0.1\). ¬øQu√© efecto tiene el par√°metro \(\alpha\)?

AR1 <- function(x_0, N, alpha, sigma) {
  x <- vector()
  for (i in seq(N)){
    # x[1] <- x_0
    if (i == 1){
      epsilon <- rnorm(1, mean = 0, sigma)
      x[i] <- alpha*x_0 + epsilon
    } else {
      epsilon <- rnorm(1, sigma)
      x[i] <- alpha*x[i - 1] + epsilon
    }
  }
  return(data_frame(AR1 = x, N = seq(N), alpha = alpha, sigma = sigma))
}

alpha <- c(0.95, 0.5)
sigma <- c(1, 0.1)
N <- c(500, 500)
x_0 <- c(0, 0)

args <- list(x_0, N, alpha, sigma)

pmap_dfr(args, AR1) %>% 
  ggplot(aes(N, AR1)) +
  geom_line(aes(color = factor(alpha))) +
  scale_color_discrete(name = "Alpha")
